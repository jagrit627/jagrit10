Computer System

Introduction

A computer system integrates hardware and software to process data efficiently. Hardware includes tangible components like the CPU, while software comprises instructions that drive these components. This chapter explores the synergy between these elements, laying the foundation for understanding modern computing.

Hardware Components

Hardware is the physical backbone of a computer system, each part serving a unique purpose:

  • CPU: The Central Processing Unit, or "brain," executes program instructions via the fetch-decode-execute cycle. Modern CPUs (e.g., Intel i5) feature multiple cores for multitasking.
  • Memory (RAM): Random Access Memory temporarily stores data for quick CPU access. For example, 8GB RAM supports smooth multitasking in a school PC.
  • Storage: HDDs offer high-capacity, cost-effective storage, while SSDs provide speed. A 256GB SSD might store your OS, boosting boot times.
  • I/O Devices: Input devices (e.g., keyboards) send data to the CPU, while output devices (e.g., monitors) display results.
Diagram: Computer Hardware Layout

Software Components

Software directs hardware operations and is divided into two categories:

  • System Software: Operating systems like Windows manage resources, enabling software-hardware interaction.
  • Application Software: Programs like Microsoft Word perform specific tasks, relying on the OS.

This distinction aids in troubleshooting and system optimization.

Data Representation

Computers use binary (0s and 1s) to represent data:

  • Numbers: Binary encodes integers (e.g., 5 = 101).
  • Text: ASCII assigns binary codes to characters (e.g., A = 65).
  • Images: Pixels store color values in binary.

Understanding this is key to data processing and storage concepts.

Number System and Boolean Logic

Number Systems

Computers use binary, but humans often employ decimal, octal, and hexadecimal systems:

  • Binary: 10 = 1010 (8+2).
  • Octal: 10 in decimal = 12 in octal.
  • Hexadecimal: 10 = A in hex, used in color codes (e.g., #FFFFFF).
Decimal 25 to Binary:
25 ÷ 2 = 12 r1
12 ÷ 2 = 6 r0
6 ÷ 2 = 3 r0
3 ÷ 2 = 1 r1
1 ÷ 2 = 0 r1
Result: 11001

Boolean Algebra

Boolean algebra uses AND, OR, and NOT operations for logical processing:

  • AND: True if both inputs are true.
  • OR: True if at least one input is true.
  • NOT: Inverts the input.

It’s foundational for circuit design and programming logic.

Logic Gates and Truth Tables

Logic gates implement Boolean operations:

  • AND Gate: 1 AND 1 = 1.
  • OR Gate: 0 OR 1 = 1.
  • XOR Gate: 1 XOR 0 = 1.

Truth tables map inputs to outputs, e.g., AND gate: 1,1 → 1; 0,1 → 0.

Computer Software and Operating System

Types of Software

Software is categorized based on its function:

  • System Software: Manages hardware (e.g., Linux).
  • Application Software: User-focused tools (e.g., Excel).
  • Utility Software: Maintenance tools (e.g., antivirus).

Operating Systems

An OS is system software that:

  • Manages memory and processes.
  • Facilitates hardware communication.
  • Examples: Windows 10, Ubuntu.

Windows dominates desktops, while Linux excels in servers.

Functions and Features

OS features include:

  • Multitasking: Running multiple apps.
  • File Management: Organizing data.
  • Security: User authentication.

Understanding OS roles enhances system efficiency.
